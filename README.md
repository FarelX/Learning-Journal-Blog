# 📝 Learning Journal Blog

A responsive personal blog built to document my journey through the Full Stack Developer Career Path on Scrimba.

---

## 📖 About the Project

This project is a responsive multi-page blog built as a learning journal. It features a dynamic blog feed, an about page, and individual post pages — all powered by vanilla JavaScript. The goal was to create a clean, readable, and accessible platform to share my learning journey while applying core web development skills.

## 🔥 Highlights

- 📱 Fully responsive design that works on any screen size.
- 🍔 Custom animated hamburger menu for mobile navigation.
- ✨ Scroll-triggered blog card animations using IntersectionObserver.
- 🗂️ Dynamic blog post rendering from a JavaScript data array.
- 🧭 Multi-page layout with shared header, footer, and navigation.
- 📄 Dedicated post page and about page with consistent styling.

## 🧠 What I Learned

- **CSS Grid:** Using `grid-template-columns` with responsive breakpoints to build a flexible blog card layout that shifts from 1 → 2 → 3 columns.
- **Responsive Design:** Building a fully responsive multi-page site that adapts cleanly to desktops, tablets, and mobile devices.
- **Mobile-First Workflow:** Designing for small screens first, then progressively enhancing for larger viewports using media queries.
- **Hamburger Menu:** Creating an animated mobile navigation menu with CSS transitions and toggling classes via JavaScript event listeners.
- **DOM Manipulation:** Dynamically generating and injecting HTML from a JavaScript data array using `.map()` and `innerHTML`.
- **IntersectionObserver API:** Implementing scroll-triggered animations on blog cards for a polished, performant reveal effect.
- **CSS Animations & Transitions:** Using `opacity`, `transform`, and `transition` to animate UI elements smoothly.
- **Relative Units:** Applying `rem` units throughout for scalable, consistent typography and spacing.
- **Media Queries:** Using breakpoints (550px, 767px, 1024px, 1200px) to control layout at different viewport sizes.
